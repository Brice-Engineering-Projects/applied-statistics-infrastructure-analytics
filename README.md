## Overview

**Applied Infrastructure Analytics** is a hands-on learning project that teaches statistics, probability, and data analysis through realistic civil and environmental engineering datasets.

Rather than relying on traditional textbook examples, this repository uses engineering scenarios involving hydrology, water distribution systems, wastewater collection systems, infrastructure reliability, and asset management.

Each lesson combines statistical theory with practical Python implementation, emphasizing concepts that are directly applicable to engineering practice and infrastructure analytics.

---

## Project Goals

The primary objectives of this project are to:

* Develop a strong conceptual understanding of statistics and probability.
* Apply statistical methods to realistic engineering datasets.
* Reinforce Python programming through practical exercises.
* Build reusable statistical analysis tools.
* Demonstrate engineering problem solving using modern data science workflows.

## Philosophy

This repository emphasizes understanding concepts before memorizing formulas.

Every lesson begins with a practical engineering problem and develops the statistical tools needed to analyze it. The focus is on answering engineering questions such as:

* How often should this event be expected?
* How likely is system failure?
* What level of uncertainty exists in the data?
* What level of risk is acceptable?
* How should statistical results influence engineering decisions?
